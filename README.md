# Sales Data Analysis using Python

This project performs exploratory data analysis (EDA) on a real-world sales dataset to identify revenue trends, product performance, and business insights.

---

## Objective
To analyze sales data and extract meaningful insights that support data-driven decision making.

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib

---

## Key Analysis Performed
- Calculated net revenue using quantity and discount
- Analyzed monthly sales trends
- Evaluated product-wise and category-wise performance
- Identified high and low revenue periods
